# Panduan Deployment TeleBot Panel Pro

- Akun [Netlify](https://netlify.com)

## 2. Deploy ke Netlify

### Metode 1: Deploy via Netlify CLI
```bash
# Instal Netlify CLI
npm install -g netlify-cli

# Login ke Netlify
netlify login

# Deploy
netlify deploy --prod
